Logging¶

Collection of classes and functions to configuration Python Logger objects, re-purposed to provide detailed status messages during model program execution.

class balsa.logging.LogFormats(value)¶

An enumeration.

class balsa.logging.ModelLogger(name, level=0)¶

Extends the standard Python Logger object, adding additional logging statements such as .report()

report(msg, *args, **kwargs)¶

Report useful model statistics or results to the user. Distinct from .info() which provides status information. Printed in green when colours are available.

tip(msg, *args, **kwargs)¶

Provide a more significant status statement (e.g. new section of the model). Similar to .info(), but more emphasized. Printed in blue when colours are available.
